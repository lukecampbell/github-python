# github-python
Simple python command-line tool to simplify interfacing with github

## Usage

```
usage: github [-h] [-b BRANCH] [-r REMOTE] [-c COMPARE] [-f FILE] [-l LINE]
              [-s SHOW] [-p [PULL]] [-o BLOB]

optional arguments:
  -h, --help            show this help message and exit
  -b BRANCH, --branch BRANCH
                        branch name
  -r REMOTE, --remote REMOTE
                        remote name
  -c COMPARE, --compare COMPARE
                        Compare branch
  -f FILE, --file FILE  show file
  -l LINE, --line LINE  Show lines from file
  -s SHOW, --show SHOW  show commit
  -p [PULL], --pull [PULL]
                        Show pull requests
  -o BLOB, --blob BLOB  Show a specific blob
```
